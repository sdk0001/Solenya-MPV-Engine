# Changelog

## 1.0.0 - 2026-04-30

- Initial public Solenya MPV Engine release.
- AIDL service compatible with Solenya plugin API version `1`.
- Replaced the previous bundled AAR with a clean local wrapper plus native libraries from fresh `mpv-android` builds.
- Bundled `arm64-v8a` and `armeabi-v7a` native stacks built from MPV, FFmpeg, dav1d, and libplacebo sources.
- Added release manifest, native build notes, checksums, and third-party notices for GitHub publication.
